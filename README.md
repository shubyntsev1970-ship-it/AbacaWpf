# ABACA: правила игры

Цель: набрать больше очков, чем соперник.

Игроки ходят по очереди и записывают результат в первую свободную ячейку выбранной строки таблицы.
За ход игрок может сделать до трех бросков. Нажмите `СТАРТ/СТОП` или `Space`.
После остановки можно кликом фиксировать нужные кубики; зафиксированные кубики не перебрасываются. Комбинацию можно выбрать мышкой в нижней строке или кликом по ее названию в таблице.

## Начисление очков

В клетки комбинаций, кроме школы, записывается сумма значений кубиков, которые составляют соответствующую комбинацию.

Примеры:

- пара `66` дает `12`;
- две пары `55` и `44` дают `18`;
- тройка `555` дает `15`;
- фул `55566` дает `27`;
- малый стрит `1-2-3-4-5` дает `15`;
- большой стрит `2-3-4-5-6` дает `20`;
- сумма `66543` дает `24`.

Для каре к сумме четырех одинаковых кубиков добавляется `20` очков.
Например, каре `5555` дает `40`.

Для абака к сумме пяти одинаковых кубиков добавляется `50` очков.
Например, абак `66666` дает `80`.

Любая комбинация, кроме школы, удваивается, если она записана с первого броска.
Школа не удваивается.

## Школа

В школе 1-6 считается количество выбранного номинала.
Три одинаковых дают `0`; меньше трех записывает минус, больше трех - плюс.
Отрицательная школа влияет на общий счет по старым правилам Abaca.

Если в кубиках нет нужного номинала, вычеркивать строку школы нельзя, пока есть свободные клетки вне школы.

## Комбинации

- Пара.
- Две пары.
- Тройка.
- Фул.
- Малый стрит `1-2-3-4-5`.
- Большой стрит `2-3-4-5-6`.
- Каре.
- Абак.
- Сумма.

Две пары считаются через полные пары: для каждого номинала берется `количество / 2`.
Например, `55566` можно записать как две пары: `55` и `66`.
`45556` не является двумя парами, потому что там только одна полная пара.

Если игрок вручную с первого броска выбирает несобранную комбинацию, игра показывает предупреждение и просит подтвердить вычеркивание.
После второго и третьего броска подтверждение не показывается.
Компьютер может вычеркивать комбинации сам, потому что это часть логики ИИ.

## Призы

Заполненная строка таблицы дает приз.
В школе приз равен номиналу строки x 5, в остальных строках таблицы - максимуму строки.

Заполненная колонка дает максимум колонки.
Приз соперника в той же строке таблицы или колонке вычеркивается, кроме последней колонки.

## Горячие клавиши

- `Space` - старт / стоп броска.
- `Left` / `Right` - выбрать кубик слева / справа.
- `Up` - зафиксировать выбранный кубик.
- `Down` - снять фиксацию выбранного кубика.

## Комбинации по клавишам

- `1` - школа 1.
- `2` - школа 2.
- `3` - школа 3.
- `4` - школа 4.
- `5` - школа 5.
- `6` - школа 6.
- `P` - пара.
- `D` - пары.
- `T` - тройка.
- `F` - фул.
- `E` - малый стрит.
- `S` - большой стрит.
- `C` - каре.
- `A` - абак.
- `+` - сумма.

Комбинацию можно выбрать мышкой в нижней строке или кликом по названию строки в таблице.
