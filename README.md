ABACA: правила игры

Цель: набрать больше очков, чем соперник. 

Игроки ходят по очереди и записывают результат в первую свободную ячейку выбранной строки.
Ход: у игрока до трех бросков. Нажмите СТАРТ/СТОП или Space. 
После остановки можно кликом фиксировать нужные кубики; зафиксированные кубики не перебрасываются.

Школа 1-6: считается количество выбранного номинала. Три одинаковых дают 0; меньше трех записывает минус,
больше трех - плюс. Отрицательная школа влияет на общий счет по старым правилам Abaca.

Комбинации: пара, две пары, тройка, фул, малая строка 1-2-3-4-5, большая строка 2-3-4-5-6, каре, абак и сумма.
Комбинация, собранная с первого броска, удваивается.
Призы: заполненная строка дает приз. В школе приз равен номиналу строки x 5, в остальных строках - максимуму строки. 
Заполненная колонка дает максимум колонки. 
Приз соперника в той же строке или колонке вычеркивается (за исключением последней колонки).

Горячие клавиши:
Space        - старт / стоп броска
Left / Right - выбрать кубик слева / справа
Up           - зафиксировать выбранный кубик
Down         - снять фиксацию выбранного кубика
